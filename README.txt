Trang web tĩnh "Trang Gia Tộc Họ Nguyễn – Hòa Yên" (offline + online)

Hướng dẫn sử dụng:
- Bản offline: Giải nén thư mục 'gia_pho_website' rồi mở file index.html bằng trình duyệt.
  Lưu ý: một vài trình duyệt (Chrome) có thể chặn fetch() từ file://; nếu ô tìm kiếm báo lỗi, hãy mở bằng trình duyệt khác hoặc dùng một server nhỏ:
    python3 -m http.server   (mở tại thư mục gia_pho_website) và truy cập http://localhost:8000

- Bản online:
  + Upload toàn bộ thư mục (không nén) lên GitHub Pages hoặc bất kỳ hosting tĩnh nào (Netlify, Vercel).
  + Hoặc upload .zip lên Google Drive và chia sẻ; để chạy trực tiếp từ Drive, có thể dùng Google Drive web preview (không đảm bảo JS fetch hoạt động).
  + Mình khuyến nghị dùng GitHub Pages: tạo repo, push nội dung, bật Pages (branch main / root).

Cập nhật tài liệu (admin):
- Để gắn link tải file Word/PDF: upload file lên Google Drive, copy share link, rồi sửa file tu-lieu.html để thay văn bản "Admin chưa gắn link tải" bằng <a href="LINK">Tải về</a>.

Nếu bạn muốn, mình có thể:
- Tạo file ZIP sẵn để bạn tải (nếu bạn cho phép upload/cho mình quyền, hoặc bạn gửi cách nhận file).
- Hướng dẫn từng bước upload lên GitHub Pages hoặc Google Sites.
